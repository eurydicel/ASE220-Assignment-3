# ASE220-Assignment-3

a multi‑page portfolio site using React, Vite, and react‑router‑dom.

## Installation
- download this repository
- open a terminal in the corresponding folder
- run "npm install"
- run "npm run dev"
- go to the localhost port provided in terminal to see the website

## Routes
- "/" = The home page
- "/contact" = The contact page with my relevant information
- "/projects" = The projects page with projects I have completed in the past
- "/experience" = The experience page with my relevant work experience

## Quotes Widget
The quotes widget on the experience page is a component made with the Node FetchAPI and useEffect. It is able to serve any number of quotes by taking a number query and getting that number of random quotes from an API using Fetch.
